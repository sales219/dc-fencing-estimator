# DC Fencing Estimate Calculator

ZIP-ready WordPress plugin for DC Fencing LLC.

## Installation

1. Zip the `dc-fencing-estimate-calculator` directory.
2. In WordPress, go to **Plugins > Add New > Upload Plugin**.
3. Upload and activate the ZIP file.
4. Add the shortcode `[dc_fencing_estimator]` to the `/online-estimates` page.

## Features

- Mobile-first customer estimate form.
- Server-side PHP estimate calculation.
- One rounded preliminary estimate shown to the customer.
- Admin and customer email notifications.
- Lead storage in the WordPress admin under **Fence Leads**.
- Lead status tracking: New, Contacted, Quoted, Won, Lost.
- Administrator-only settings for pricing, adjustments, rounding, notifications, customer email copy, and disclaimer text.
